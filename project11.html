USB Controller



Nummer :

11

Plattform :

Arduino

Schlagwörter :

Controller Prototyping USB

Drehknöpfe, Schieberegler, Mikrofon und Ultraschall-Sensor senden über einen Mikrocontroller die Daten via USB-Verbindung an einen Computer. Dabei fungiert der programmierte Mikrocontroller als Steuereinheit, durch den das gesendete Signal in Echtzeit am Empfänger verarbeitet werden kann.





Zwei OLED Displays zeigen dabei die aktuelle Position der Drehknöpfe als Quadrat bzw. Kreis an und der Schieberegler steuert die Sensitivität der Sensoren.



c Simon Penn


Der Controller soll in einem kompakten Format Möglichkeiten von Sensoren und Aktuatoren aufzeigen. Gleichzeitig ist die Box ein portables Toolkit, um neue Ideen bei interaktiven Elementen in Museen möglichst unkompliziert zu entwickeln, testen und kalibrieren.



c Simon Penn
