Image Processing Pixels Algorithm

Mit der Entwicklungsumgebung Processing wurde ein Code geschrieben, mit dem ein beliebiges Bild zu einem bewegten Bild transformiert wird. Während des Ladevorganges wird das Bild Pixel für Pixel analysiert und entsprechend eines Algorithmus mit Punkten, deren Größe der Helligkeit des ursprünglichen Pixels entspricht, neu gezeichnet.

Aktuell können hierfür wahlweise folgende zwei Algorithmen genutzt werden:

  1. Das Bild wird 5-mal in der Sekunde geladen, wobei die Koordinaten der Punkte nach jeder Iteration um einen Zufallswert verschoben werden. Durch das minimale Versetzen der Punkte entsteht ein oszillierendes Bild, das an einen Ameisenhügels im Wald erinnert.





  1. Das Bild wird in eine 3-dimensionale Maske transformiert, wobei die Z-Werte der Punkte durch die Helligkeit der Pixel definiert werden. Das Drehen um die Y-Achse lässt die Betrachter:innen einen Blick hinter die Maske werfen und lässt dabei die Grenze zwischen Innen- und Außenperspektive verblassen.





Eine Kamera schießt nach Drücken eines Auslösers ein Foto, welches automatisch in “Processing” geladen wird. Anschließend wird in der Entwicklungsumgebung mit dem ausgewählten Algorithmus eine Animation erstellt, welche via E-Mail oder Social Media verschickt werden kann.
